# Aplicación de Álgebra Lineal en Salud

## Descripción
Este proyecto implementa operaciones básicas con vectores utilizando Python y la biblioteca NumPy.

Se desarrollan:
- Suma de vectores
- Resta de vectores
- Producto por escalar

## Aplicación
Se modela una situación del área de la salud en la cual se combinan dos medicamentos para obtener una dosis específica.

Esto se representa mediante un sistema de ecuaciones lineales:

2x + y = 5  
x + 3y = 7  

La solución indica la cantidad necesaria de cada medicamento.

## Tecnologías usadas
- Python
- NumPy
